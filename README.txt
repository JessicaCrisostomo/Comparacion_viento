INSTRUCCIONES

1) Copia tus figuras mensuales dentro de la carpeta figuras/.
   Deben llamarse:
   serie_01.png
   serie_02.png
   ...
   serie_12.png

2) Reemplaza los archivos CSV de ejemplo ubicados en datos/:
   intensidad.csv
   direccion.csv

3) Para abrir la página:
   - Opción simple: doble clic en index.html.
   - Si las tablas CSV no cargan por seguridad del navegador, abre la carpeta con VS Code y usa Live Server.

4) Estructura esperada:
   web_viento/
   ├── index.html
   ├── style.css
   ├── script.js
   ├── figuras/
   └── datos/
